# 影像店营业

## 这是什么

绝区零录像店「Random Play」(澄辉坪)的**日常经营玩法**(无战斗)。玩家每日选宣传员 + 上架匹配主题的录像带 + 开始营业,赚取**丁尼**(经营收益)。

## 玩法目标

每日完成营业流程(选宣传员 → 上架录像带 → 开始营业),赚取丁尼。**已营业(正在营业)则跳过**。

## 核心机制

- **宣传员**:派驻代理人(12 选 1-2),影响营业效果。
- **录像带主题**:营业需 3 个主题(喜剧/惊悚/访谈等 16 选 3);上架匹配主题的录像带提升效果。
- **营业状态**:`正在营业`(已营业,跳过)/ 未营业(走流程)。

## 子玩法(游戏含义)

- **选宣传员**:派驻代理人(脚本按 config 名称选,OCR 优先 + 头像兜底)。
- **上架录像带**:为 3 主题各上架匹配录像带(主题筛选 + 模糊匹配 16 主题)。
- **开始营业**:确认营业 → 营业后确认 → 完成。
