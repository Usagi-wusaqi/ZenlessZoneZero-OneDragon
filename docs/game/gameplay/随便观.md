# 随便观经营

## 这是什么

绝区零 **2.0**(2025-06-06)的**永久经营玩法**,位于卫非地半岛澄辉坪,是云岿山的据点与训练场。玩家接管荒废的随便观道观,通过派遣邦布(劳动力)游历采集、制造货品、售卖获利、完成邻里委托,赚取**云纹徽**提升**复兴度**,把随便观恢复往日辉煌。主线第二章「云霞同归处」的「随便观复兴计划」解锁。(据 Game8 wiki / 攻略)

## 玩法目标:复兴度

提升**复兴度(Restoration Level,1→45 级)**。每级有复兴目标(制造指定货品 / 售卖赚云纹徽 / 开游历副本 / 完成饮茶仙委托 / 持有指定等级邦布),达标升复兴度、解锁内容。

**解锁里程碑**(据 Game8):
- Lv.3 饮茶仙 | Lv.8 好物铺 | Lv.10 德丰大押 | Lv.15 邦巢(雇更多邦布)
- Lv.5 起可买邦布 | Lv.26 首只 S 级邦布 | Lv.35 复兴任务「称愿笺」阶段
- **Lv.45 完全复兴 → 解锁自动托管**(见下)

> 百通宝周期获取上限 **5000/5000**(实拍饮茶仙);云纹徽需求随复兴度急剧增长(Lv.45 累计需上千万,据 Game8)。

## 经济循环(核心)

```
邦巢雇邦布(劳动力,三维:经营/游历/续航)
    │
    ├─游历──产材料──→ 制造坊──造货品──→ 售卖铺──赚云纹徽──→ 升复兴度
    │
    └─饮茶仙委托──百通宝/云纹徽/称愿笺──→ 德丰大押(换菲林) / 好物铺(换道具/插件)
```

**主循环**:游历产材料 → 制造成货品 → 售卖换云纹徽 → 升复兴度。
**辅循环**:饮茶仙委托 + 德丰大押/好物铺,把经营产出转成游戏通用奖励(菲林/养成材料)。

## 邦布系统(劳动力核心)

邦布是随便观的劳动力,有**三维属性**(实拍邦巢卡片,据 Game8 对应英文):
- **经营(Acumen)**:制造能力(制造岗邦布高,实拍 S 级达 316)
- **游历(Adventure)**:游历能力(游历岗邦布高)
- **续航(Battery Life)**:电量(派驻消耗,实拍 120/158)

**岗位**由天赋决定:**制造 / 售卖 / 游历** 三类。等级 B/A/S,Lv.1→30,喂「邦布能源插件」升级,特定等级解锁天赋。
- **制造 / 售卖岗**:优先 **S 级**(属性不对口也上;据攻略 S 级制造约每 5-7 秒一件、售卖约每 12 秒一件)。
- **游历岗**:优先**对应属性**邦布(50% 效率加成)。
- **来源**:教程给初始 B 级;**邦巢**花百通宝雇(S/A/B 抽卡式刷新)。

## 托管 vs 手动(核心机制)

**自动托管**是 **2.4 版**引入,需**复兴度满 45(完全复兴)**才解锁(据 Game8;实拍入口「LV.45 完成复兴 / 自动托管中」对上):
- **托管**:一键挂机,自动跑经营收益(**实拍:3,031,200 云纹徽/轮**,自动售卖产),**不消耗材料/邦布**,只挂时长。入口 tab 锁,不能手动操作。**定位 = 低保挂机**(解放双手但收益有限)。
- **手动**:停托管后,手动调度各子玩法(耗材料,精细优化邦布/货品/委托),**收益显著高于托管**(据攻略:复兴度 45 级后跟随售卖趋势,约每小时 13 万云纹徽;两周可获约 4368 万,够搬空周期商店约 3000 万)。
- **互斥**:托管中入口 tab 锁,不能手动派遣/制造/售卖。
- **选择**:懒人低保用托管;冲收益最大化 / 搬空周期商店 / 刷复兴目标用手动。

> ⚠️ **版本变迁**:2.4 版引入托管,2.5 版调整了周期奖励(社区有争议),收益环境以当前游戏实际为准。

## 子玩法(游戏含义)

- **游历**:派邦布小队去澄辉坪各区域副本(制造区 / 社区旧址 / 科研院旧址,**深处**后缀=进阶),耗时返回产材料。副本探险进度达 100% 解锁下一阶。
- **制造坊**:邦布用材料研发/制造货品(本草方剂→镇厄散→…→`[精淬]` 系列高阶)。4 制造台,复兴度越高解锁越高价值货品。
- **售卖铺**:货品上架,售卖邦布自动卖赚云纹徽;拾遗额外收益(实拍 0/30 次/日)。
- **饮茶仙**:接**定期采办**委托(交材料换百通宝/云纹徽,**周期刷新**)+ **邻里心愿**委托(产称愿笺,**固定一次性剧情委托,每玩家完成一遍即清空,非周期**)。
- **邦巢**:花百通宝雇邦布(S/A/B 抽卡式),补充劳动力分配岗位。
- **好物铺**:云纹徽兑换道具(邦布能源插件/货架/材料);也可把空洞采集材料换经营材料。
- **德丰大押**:百通宝兑换菲林/养成材料(高价值终端奖励)。

## 日常 / 周期

- 周期刷新:百通宝上限 5000 / 定期采办列表(实拍 04 天 12 时刷新)/ 好物铺周期商店(约两周)。
- 自动托管:挂时长,定期停托管领收益结算(百通宝/云纹徽)再重启。
