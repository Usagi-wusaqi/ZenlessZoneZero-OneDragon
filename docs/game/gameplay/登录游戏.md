# 登录游戏(打开游戏 → 进入大世界)

## 流程概述
从「打开游戏」画面到进入大世界(Overworld)的完整登录流程。前置:游戏启动(米哈游启动页 → 警告 → 标题页 → 打开游戏),见 [游戏启动](游戏启动.md)。

## 完整流程

### 1. 打开游戏 · loading 态
账号已缓存 → 自动「加载配置数据中」(进度 %) → ready 态。

### 2. 打开游戏 · ready 态
出现 `点击进入游戏` 按钮。

**自动登录路径**(账号已缓存,**常用**):
- 点 `点击进入游戏` → 登录服务器中 → 登录成功 → 加载画面 → 大世界。

**手动登录路径**(无缓存 / 点「切换账号」退出后):
- 点右下 `切换账号`(=退出登录) → 退出登录弹窗 → 选「保留/清除登录记录」+ 确定:
  - **保留记录** → 账号确认态(显历史账号) → `进入游戏` → 选区服(自动)→ 加载配置中 → ready → `点击进入游戏` → 登录服务器中 → ...
  - **清除记录** → 手动登录态 → 填凭证 → `进入游戏` → 选区服(自动)→ 加载配置中 → ready → `点击进入游戏` → 登录服务器中 → ...

**手动登录 3 种方式**:
- **验证码登录**(默认):手机号 + 验证码 → `发送`(点击后变灰 + 60S 倒计时,期间不可重发)→ `进入游戏`。
- **账号密码登录**(底部「账号密码」按钮切入;左上 ◀ 返回 → 验证码态):手机号/邮箱 + 密码 → `进入游戏`。
- **扫码登录(QR)**(验证码态点**对话框左上角**小二维码图标 @~(552,234)):手机扫码 → 扫码成功 → 手机点【确认登录】→ 正在登录 → 选区服(自动)。

### 3. 选区服/登录确认态(枢纽态)
退出登录后 / 任意登录方式成功(验证码/账号密码/扫码/账号确认-进入游戏)后均停此态。国服仅「新艾利都」一个服,无需选服。画面 `点击登录` 按钮**弹出登录对话框**(选账号 + 验证码/账号密码/扫码);完成登录后对话框消失,选区服**自动过渡**到加载配置中(loading)→ ready → 再点 `点击进入游戏` 进大世界。退出登录后游戏在选区服自动点 `点击登录` + 完成登录 + 过渡到加载,故常观察为"不用点";close-X 登录对话框则停在选区服手动态,需手动点 `点击登录`。

### 4. 登录服务器中态 → 登录成功态
`登录游戏服务器中,请等待`(自动,无交互) → `登录成功` toast(短暂)。

### 5. 加载画面(lore)
通用加载画面(左侧世界观 tip 轮换 + 右侧场景插画),自动推进。无固定特征,**不适合建 screen_info**。

### 6. 大世界(Overworld)
进入游戏主画面。如果上次退出时在某玩法中(如迷失之地/特遣调查),弹出「特遣调查是否继续?」确认弹窗。

## 条件分支总结

| 条件 | 路径 |
|---|---|
| 账号已缓存(自动登录) | ready → `点击进入游戏` → 登录服务器中 → 登录成功 → 加载 → 大世界 |
| 退出登录 · 保留记录 | ready → 弹窗 → 保留 → 账号确认 → `进入游戏` → 选区服(自动)→ 加载 → ready → `点击进入游戏` → 登录服务器中 → ... |
| 退出登录 · 清除记录 | ready → 弹窗 → 清除 → 手动登录(验证码/密码/扫码) → `进入游戏` → 选区服(自动)→ 加载 → ready → `点击进入游戏` → ... |
| 全新设备(无缓存) | 直接 → 手动登录态 → ... |

## 自动化对应
项目的 `open_and_enter_game` operation 走**自动登录路径**。关键画面:
- `打开游戏` ready 态(点 `点击进入游戏`)。
- 登录服务器中 / 登录成功(等待过渡)。
- 加载画面(等待)。
- 大世界(到达 = 成功)。
