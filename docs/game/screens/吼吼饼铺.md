# 吼吼饼铺(每日签到盲盒)

吼吼饼铺每日「零食盲盒」签到(`hou_hou_bakery` app)。**纯签到(无战斗)**,与刮刮卡/卦象集录同类,每日刷新后最多领一次。

## 入口 + 状态流转

大世界 → `Transport(布亚斯特城区, 吼吼饼铺)` → `interact(F)` 吼吼先生 → 领奖界面 → 领取盲盒 → 返回大世界。

流转(`HouHouBakeryApp` `@operation_node` 链):
传送 → 移动交互(传送点已近,无需 move,interact 开领奖界面) → 领取奖励(单节点自循环 OCR 分发)→ 返回大世界。

`collect` 领取奖励 OCR 分发:
- 「同类型奖励」: 领取终点(今日已领取同类型 / 已在刮刮卡·卦象集录等领过)→ 退出。⚠️ 必须在「确定/确认」前判,防弹窗自带确认被误点 → 关弹窗回集卡机 → 再点盲盒 → 又弹窗,循环耗尽重试。
- 「确定/确认」: 卡片揭示 → 点确认领 + 标 `claimed`。
- 「查看今天」: 盲盒已居中 → 点屏幕中心开盒。
- 「每日可领取一次」: 集卡机可领 → 点 `盲盒` area。
- 其余(黑屏过渡/加载)→ 重试。
