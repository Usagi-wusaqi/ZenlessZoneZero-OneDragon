# 委托助手(对话/剧情/钓鱼辅助循环器)

`commission_assistant` 是**辅助循环器**(非固定游戏画面):用户手动玩剧情/钓鱼/战斗,app 辅助点对话选项/跳过剧情/钓鱼指令/自动战斗。含可选 auto_battle(边界)。

## 定位(非普通 app)

委托助手**不是自动跑流程的 app**,是**辅助循环器** —— 用户手动推进剧情/钓鱼/战斗,app 监听键盘(`dodge_switch`/`auto_battle_switch` 切模式)+ 自动点对话选项/跳过剧情/钓鱼指令/自动战斗。

`run_mode`:0=对话(默认)/ 1=闪避 / 2=自动战斗。

## 辅助逻辑(`CommissionAssistantApp`)

- `dialog_mode`:对话框确认 click / 战斗画面交互 / 二级菜单等待 / 空洞处理(`hollow_event_utils`)/ 剧情检测。
- `story_mode`(AUTO):`中间选项区域` click(模板定位 Y 轴,模板位置点击无反应)/ 「自动」等待 / 菜单展开切自动。
- `story_mode`(SKIP):「跳过」click + 确认框 / 菜单·自动遮罩后处理;点菜单后 5s 内不处理对话框(防动画误点)。
- `auto_mode`(run_mode 1/2):闪避(`dodge` 配置)/ 自动战斗(`auto_battle` 配置,`check_battle_state`)。
- `check_fishing`:钓鱼(抛竿/等待上鱼/时机上鱼/连点/长按,`钓鱼` screen)。
- `check_knock_knock`:短信(点最下方文本 / 「以上为最新」→ 关闭)。
- 选项点击保护:`CHOSEN_OPT_HOLD_SEC`(0.5s 内连点) / `CHOSEN_OPT_MAX_SEC`(2s 误识别兜底)。
