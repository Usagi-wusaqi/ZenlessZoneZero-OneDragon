# 卦象集录

与澄辉坪 NPC 阿朔交互后打开的「卦象集录」每日玩法界面(每日一次,与刮刮卡 / 吼吼饼铺等**同类活动共享领取次数**)。

## 何时出现 + 状态流转

入口:大世界 → `Transport(澄辉坪, 阿朔)` 到位 → `interact(F, press_time=0.2)` → **卦象集录主界面**。

| 子态 | 入口 | 出口 |
|---|---|---|
| 1. 主界面 | `interact(F)` 打开 | 点「区域-获取卦象」("開"字图标)→ 滑动态 / 已领取提示 |
| 2. 滑动获取卦象 | 主界面点获取卦象(当日首次) | `drag_to` 来回拖满 → 领奖确认 |
| 3. 领奖确认 | 滑动完成 | 点「确认」→ 返回大世界 |
| 4. 今日已领取终点 | 主界面点获取卦象(当日已在刮刮卡等同类领过) | 点「确认」→ 返回大世界 |

> 2 / 3 子态(滑动获取卦象 + 领奖确认)**待补** —— 当日已领取看不到滑动流程(2026-07-12 实测今日已在刮刮卡领过,点获取卦象直出「今日已领取」提示)。

## 识别特征(稳定锚点)

- 文字「卦象集录」(主界面底部,红 / 黑色书法字)。
- 「開」字圆形木质主图标(区域-获取卦象)。
- 易变:卦象符号环绕(随机)、当日状态(可领 / 已领)。

## 可交互元素

- **区域-获取卦象**(图形按钮,screen_info area 纯坐标 pc_rect `1318,298 - 1650,640`,中心 ~`1484,469`):「開」字圆形主图标,点击触发获取卦象(当日首次 → 滑动;已领 → 提示)。
- **确认**按钮(今日已领取弹窗,~`950,609`):文字按钮,OCR「确认」。
- 左上角**返回**(通用「返回」模板,~`92,22`)。
