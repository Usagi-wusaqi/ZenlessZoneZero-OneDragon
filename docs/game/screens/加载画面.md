# 加载画面(lore 加载过渡)

## 何时出现
登录成功后 → **本画面** → 大世界/主菜单。游戏内的**通用加载画面**,每次加载时出现。左侧 lore 文字(世界观 tip)**会轮换**(每次不同),右侧是静态场景插画。

## 识别特征(稳定锚点)
- **左侧 lore 区域**:标题(如 `【港口工厂旧址】`)+ 描述文字 —— **内容动态轮换**,勿当特征。
- **右侧场景插画**:静态(对应 lore 的场景图)。
- **UID**:右下角(所有游戏内画面通用)。
- **无 `NOW LOADING` 文字**(那是另一个加载态 `加载中`/`loading.yml`,见备注)。

## 可交互元素
**无** —— 纯过渡加载画面,自动推进。
