# 地图

> 普通传送(`MapTransport`)的 **2D 列表传送画面**。**传送机制**(两种地图区别 / 入口按 `N` / 传送点类型 / 传送行为)见 [mechanics/传送](../mechanics/传送.md);本 doc 记画面识别 + 本画面传送流程。

## 子态流转

| 子态 | 入口 | 出口 |
|---|---|---|
| ⓪ 区域选择 default 态 | 按 `N` 进(见 mechanics) | 点区域(澄辉坪/…)→ ① |
| ① 选传送点(列表) | 选区域后底部出传送点列表(随便观/阿朔/汀曼咖啡…) | 点传送点 → ② |
| ② 传送确认弹窗 | 点传送点 → 弹「是否传送至该地点？」(确认/取消) | 点「确认」(`地图/确认` area)→ 传送落地;取消 → 回① |

> `MapTransport` 流程(`src/zzz_od/operation/map_transport.py`):选区域(`choose_area`,整屏 OCR 区域名 + 横向 drag)→ 选传送点(`choose_tp`,OCR `地图/传送点名称` 列表)→ 点确认(`click_tp`,`地图/确认`)。

## 识别特征(稳定锚点)

- **传送点名称** area(@70,1000-1848,1058,底部传送点列表,无 text)。
- **确认** area(@1024,600-1232,708,text`确认`,弹窗绿√按钮)。
- **左上角返回**(template `menu/back` @82,13-150,90)。
- **易变值**:传送点名称 / 当前选择(**动态**)。

## 可交互元素

- 传送点列表(点传送点 → 弹确认)。
- 确认 / 取消(确认弹窗)。
- 左上返回。

## 识别快照

### 0. 区域选择 default 态(`区域选择.webp`,2026-07-18 实拍)

按 `N` 后出**整屏区域列表**(澄辉坪/六分街/斯科特哨站/热望角/光映广场/奥蕾莉亚学院/[空洞]航天城/…),横向排列 + drag 切换。点区域(澄辉坪 `526,592`)→ 底部出传送点列表(①)。**无独立 screen_info**(整屏 OCR,误匹配菜单/3D地图)。

### 1. 选传送点态(`选传送点.webp`,2026-07-15 实拍)

点区域(澄辉坪)后底部出传送点列表。OCR:随便观 / 阿朔 / 汀曼咖啡 / 录像店营业点 / 饮茶仙 / 临界推演(`传送点名称` area @70,1000-1848,1058)。点传送点 → ②。

### 2. 传送确认弹窗态(`传送确认弹窗.webp`,2026-07-15 实拍)

点传送点后弹「是否传送至该地点？」+ 确认(绿√ @1101,609,`地图/确认` area @1024,600-1232,708)+ 取消(红× @798,610)。点确认 → 传送落地。
