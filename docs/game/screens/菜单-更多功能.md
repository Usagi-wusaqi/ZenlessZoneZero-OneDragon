# 菜单-更多功能

## 何时出现 + 状态流转

菜单底部点「更多」tab → **菜单-更多功能**(模态)。功能入口枢纽,多个 app 经此进入(预备编队 / 兑换码 / 资源管理 / 登出…)。

| 出口 | 动作→下一态 |
|---|---|
| 功能按钮 | 点对应功能 → 各功能画面(兑换码→兑换码输入;登出→登出确认;预备编队→…) |
| 关闭X / 返回 | → 菜单 |
