# 打开游戏(登录页)

## 何时出现
启动序列(米哈游启动页 → 警告:游戏前详阅 → 绝区零标题页)之后出现的登录页。自动登录路径下两个子态:
- **loading 态**(_1783242291962):账号已缓存,进入后自动「加载配置数据中」(进度 %),无进入按钮。
- **ready 态**(_1783242299811,本基线):加载完出现 `点击进入游戏` 按钮,点此进入游戏。
- **手动登录态**(未截图,**待补**):无缓存账号 → 账号密码输入 + 进入游戏。

- **`点击进入游戏` 按钮**(中下 @(892,873,137×27),ready 态)—— 主交互。
- **版本号**(右下)。

- **右边沿图标按钮**:4 个无文字图标按钮(资源检测/设置/游戏公告/切换账号),OCR 看不见,用传统 CV(HoughCircles)定位 + 裁模板建 `screen_info` template area(模板目录 `assets/template/enter_game/`)。
