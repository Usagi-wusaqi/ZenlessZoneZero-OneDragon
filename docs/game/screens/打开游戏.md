# 打开游戏(登录页)

## 何时出现
启动序列(米哈游启动页 → 警告:游戏前详阅 → 绝区零标题页)之后出现的登录页。多个子态(同一 `screen_name=打开游戏`):
- **loading 态**(_1783242291962):账号已缓存,进入后自动「加载配置数据中」(进度 %),无进入按钮。
- **ready 态**(_1783242299811,本基线):加载完出现 `点击进入游戏` 按钮,点此进入游戏(自动登录)。
- **退出登录弹窗态**(_1783247876257):ready 态点右下「切换账号/退出登录」按钮后弹出的确认弹窗,选「清除/保留登录记录」→ 确定/取消。
- **账号确认态**(_1783247887600):退出登录弹窗选「保留登录记录」+ 确定 后出现(**条件:曾有登录且保留登录状态**);显手机号账号 + `进入游戏` + `登录其他账号`。
- **手动登录态**:无缓存账号(全新设备 / 清除登录记录 / 点「登录其他账号」)进入;默认 **验证码登录态**(_1783247903033,手机号+验证码),底部可切「账号密码」/TapTap。
  - **验证码登录态**(_1783247903033):手机号 + 验证码输入 + `发送` + 两个 checkbox(保留登录记录 / 已同意协议,~12px 圆形带彩对号,**太小不建模板**)+ `进入游戏` + 底部登录方式按钮(账号密码 / TapTap)。
  - **账号密码登录态**(_1783247914584):手机号/邮箱 + 密码输入(密码框带小眼睛)+ `进入游戏` + `忘记密码` + 2 checkbox。从验证码态底部「账号密码」按钮切入。
  - **选区服/登录确认态**(_1783247920218):**枢纽中间态**,退出登录后 / 任意方式登录成功(验证码/账号密码/扫码确认/账号确认-进入游戏)后均停此态。国服仅「新艾利都」一个服,无需选服。画面有 `点击登录` 按钮——**点它弹出登录对话框**(选账号 + 验证码/账号密码/扫码);完成登录后对话框消失,选区服**自动过渡**到加载配置中(loading 态) → ready 态。所谓"不用点"=退出登录后游戏在选区服**自动点了 `点击登录`** + 完成登录 + 过渡到加载,一气呵成;若 close-X 登录对话框则停在选区服**手动态**,需手动点 `点击登录` 重开对话框。
  - **扫码登录态(QR)**(_1783247906852):验证码登录态点**对话框左上角**小二维码图标(@~(552,234),非"手机号左上角") → 弹出扫码登录(二维码 + `支持APP` 第三方图标 + 保留登录记录 checkbox)。**粗略记录,不建 area**。
    - **扫码成功态**(_1783380865638):手机扫码后,中央显示 `扫码成功` + `请在手机上点击【确认登录】`,右下有 `切换账号` 按钮;手机点确认后该按钮变 `正在登录...` 文本(按钮消失)→ 选区服/登录确认态。

## 状态流转
子态间切换(动作/条件 → 下一态):

| 当前态 | 动作 / 条件 | 下一态 |
|---|---|---|
| (启动,账号已缓存) | 自动 | loading 态(加载配置数据中) |
| loading 态 | 配置数据加载完 | ready 态 |
| ready 态 | 点 `点击进入游戏` | 登录服务器中态 |
| ready 态 | 点 `切换账号`(=退出登录) | 退出登录弹窗态 |
| 退出登录弹窗态 | 点 `取消` | ready 态(返回) |
| 退出登录弹窗态 | 选「保留登录记录」+ `确定` | 账号确认态 |
| 退出登录弹窗态 | 选「清除登录记录」+ `确定` | 手动登录态(验证码登录) |
| 账号确认态 | 点 `账号确认-进入游戏` | 选区服/登录确认态(自动推进,见下) |
| 账号确认态 | 点 `登录其他账号` | 手动登录态(验证码登录) |
| (全新设备,无账号) | 自动 | 手动登录态(验证码登录) |
| 手动登录态(验证码) | 点底部「账号密码」 | 账号密码登录态 |
| 账号密码登录态 | 点左上 ◀ 返回(`国服-返回按钮` 模板 `login_back`) | 手动登录态(验证码登录) |
| 手动登录态(验证码) | 输手机号 → `发送`(按钮变灰+60S倒计时)→ 输验证码 → `进入游戏` | 选区服/登录确认态 |
| 账号密码登录态 | 输账号+密码 → `进入游戏` | 选区服/登录确认态 |
| 手动登录态(验证码) | 点对话框左上角小二维码图标(@~(552,234)) | 扫码登录态(QR) |
| 扫码登录态(QR) | 手机扫码成功 | 扫码成功态(_1783380865638:`扫码成功`+`请在手机上点击【确认登录】`) |
| 扫码成功态 | 手机点【确认登录】 | 正在登录态(右下`切换账号`按钮→`正在登录...`文本,按钮消失)→ 选区服/登录确认态 |
| 选区服/登录确认态(枢纽态) | 点 `点击登录` | 登录对话框(选账号 + 验证码/账号密码/扫码) |
| 选区服/登录确认态 | 完成登录后**自动**(对话框消失) | 加载配置中(loading 态) → ready 态 |
| 登录服务器中态 | 服务器连接成功(自动) | 登录成功态 |
| 登录成功态 | 自动(短暂展示后) | 出打开游戏 → 加载画面(_1783247956712) |

- **验证码登录态**(手动登录默认):手机号输入框(`+86` + 占位 `输入手机号注册/登录`)+ 验证码输入框 + `发送`(验证码)@(1181,445)(**点击后按钮变灰、显示 60S 倒计时**,期间不可重发)+ `进入游戏` @(907,689)+ 2 checkbox(保留登录记录 / 已同意协议,~12px 圆形带彩对号,**太小不建模板**)+ 底部登录方式按钮(`账号密码` @(818,791) / `TapTap` @(1054,790))+ **对话框左上角小二维码图标** @~(552,234)(点 → 扫码登录态;注意是**对话框左上角**,不是"手机号左上角",在标题区左侧)。手机号 / 验证码是**动态输入**;占位符文字 + `发送` / `账号密码` / `TapTap` OCR 可识别。
- **账号密码登录态**:手机号/邮箱 + 密码输入框(右侧小眼睛)+ `进入游戏` + `忘记密码` + 2 checkbox + **左上返回按钮 ◀** —— `国服-返回按钮` area(template `login_back`,圆形 r≈23,中心 (557,241);点 → 验证码登录态)+ 右上关闭 X。账号 / 密码是**动态输入**。

## 备注
- **右边沿图标按钮**:4 个无文字图标按钮(资源检测/设置/游戏公告/切换账号),OCR 看不见,用传统 CV(HoughCircles)定位 + 裁模板建 `screen_info` template area(模板目录 `assets/template/enter_game/`)。
- **退出登录弹窗**:弹窗态可交互元素已补 area(标题-退出登录 / 2 单选项 / 取消 / 确定);选中状态(哪个 radio 绿)OCR 看不出,需颜色阈值或选中/未选中 radio 模板(**待补**)。
- **`切换账号` ≡ 退出登录**:底部那个按钮(y≈997)既是「切换账号」也是「退出登录」(同义),screen_info 沿用既有 area_name「切换账号」,不改名。
- **多子态**:本屏有 loading / ready / 退出登录弹窗 / 账号确认 / 手动登录(验证码 / 账号密码 / 扫码 QR + 扫码成功) / 选区服-登录确认 等子态,见上方「状态流转」表。
- **选区服是枢纽态(非瞬态)**:退出登录后 / 任意登录方式成功(验证码 / 账号密码 / 扫码确认 / 账号确认-进入游戏)后均停 选区服/登录确认态。它的 `点击登录` 按钮**弹出登录对话框**(选账号 + 验证码/密码/扫码);完成登录后对话框消失,选区服自动过渡到 加载配置中(loading 态)→ ready → 再点 `点击进入游戏` 才进大世界。退出登录后游戏在选区服**自动点 `点击登录`** + 完成登录 + 过渡到加载,故常观察为"不用点";若 close-X 登录对话框(选账号列表)会停在选区服**手动态**,需手动点 `点击登录` 重开对话框。
- **返回按钮 ◀**:账号密码登录态左上有圆形返回按钮(`国服-返回按钮` area / template `login_back`,圆 r≈23 中心 (557,241)),点 → 验证码登录态。验证码态左上同一位置是小二维码图标(进扫码登录),二者随子态切换。
