# 对话(大世界 NPC / 通用)

大世界 `interact(F)` NPC 触发的对话画面,或剧情 / 任务对话。**通用兜底画面**(像加载画面,无固定文字特征):NPC 名不固定,旁白对话甚至不显示 NPC 名。

## 何时出现

- 大世界 `interact(F)` NPC(代理人好感度 / 任务 NPC / 狮耶等)。
- 剧情对话 / 任务对话(主线 / 支线 / 委托)。

## 识别特征(稳定锚点 = 结构,非文字)

- **下方对话框**(固定 UI 结构,非文字)。
- **NPC 名**:在 `好感度标题` area(pc_rect `892,798-1042,836`,底部中央),NPC 名不固定(狮耶 / 代理人 / 任务 NPC);**旁白对话无 NPC 名**(该 area 空)。
- **选项**(可选):`好感度选项` area(`1420,552-1918,804`,右侧);无选项普通对话无。
- ⚠️ **无固定文字特征**(像加载画面)→ analyze 通常无精准匹配(`screens=[]`),靠对话框结构 + `好感度标题` area 判定。

## 可交互元素

- **好感度选项**(右侧 area,有选项对话):点选项推进。
- **对话框确认 / 取消**(`1036,593-1212,657` 确认 / `726,600-918,652` 取消,有按钮对话)。
- **无选项普通对话**:点「菜单-返回」退出(BackToNormalWorld `_handle_agent_dialog`:无选项 → `round_by_click_area('菜单', '返回')`)。

## 识别快照

### 1. 有 NPC 名对话(source_image: 有NPC名对话.webp)

- 匹配画面:无精准命中(analyze `screens=[]`);靠对话框结构 + `好感度标题` area 的 NPC 名(OCR「狮耶」`935,802` 在 `892,798-1042,836` 内)。
- 全量 OCR:嗯呢嗯呢呢！…(振兴随便观！…) / 狮耶(NPC 名) / 觀便随(背景招牌)。
- vision:下方对话框 + NPC 名「狮耶」+ 第 3 人称角色背部(面向 NPC)。

### 2. 旁白对话(无 NPC 名)(待补)

- 识别:对话框结构 + `好感度标题` area 空(无 NPC 名)。需现场快照。
