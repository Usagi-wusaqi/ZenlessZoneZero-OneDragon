# 3D地图

> 立体城市俯视(`world_patrol`/锄大地用)。**传送机制**(两种地图区别 / 入口按 `M` / 与普通地图区别)见 [mechanics/传送](../mechanics/传送.md);本 doc 记画面识别 + 3D 传送流程。

## TransportBy3dMap 流程

(`src/zzz_od/application/world_patrol/operation/transport_by_3d_map.py`)

| 步骤 | 节点 | 作用 |
|---|---|---|
| 1 | 初始回到大世界 | 已在 3D地图则跳过,否则 BackToNormalWorld |
| 2 | 打开3D地图 | 点大世界「小地图」area |
| 3 | 选择区域 | OCR `区域-区域列表` + 纵向 drag |
| 4 | 展开子区域列表 | 点 `按钮-当前子区域`(区域有子区域时) |
| 5 | 选择子区域 | OCR `区域-子区域列表` |
| 6 | 打开筛选 | 点 `按钮-筛选` → 等 `标题-标识点筛选` |
| 7 | 筛选传送点 | OCR `区域-筛选选项` 点「传送」/「裂隙信标」 |
| 8 | 关闭筛选 | 点 `按钮-关闭筛选` |
| 9 | 最小缩放 | `按钮-最小缩放` drag 到最小 |
| 10 | 搜索传送点循环 | `区域-地图` 上 template 匹配传送点图标 → 点图标 → 等 `按钮-前往` → OCR `标题-当前选择传送点` 匹配目标 |
| 11 | 点击前往 | 点 `按钮-前往`(until 消失)→ 加载落地 |

> ⚠️ 子区域 / 筛选 / 图标搜索 / 前往等子态截图待补(目前仅默认态 `map_3d_default` 实拍)。可按 `TransportBy3dMap` 节点逐步截图补全。
