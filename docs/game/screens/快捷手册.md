# 快捷手册

绝区零**玩法总入口**(绝大多数周期 / 常驻玩法的统一入口)。大世界右上角「快捷手册」按钮进。5 个 TAB(目标 / 日常 / 训练 / 作战 / 战术),每 TAB 列玩法卡片(玩法名 + 奖励进度 + 「前往」传送按钮)。

**为什么重要**:式舆防卫战 / 迷失之地 / 零号空洞 / 危局强袭战 / 咖啡店 / 影像店 / ... 等玩法入口都在这里;app `TransportByCompendium(tab, category, mission_type)` 经此传送。

## 何时出现 + 状态流转

- **入口**:大世界 右上「快捷手册」按钮(~1531,60,需 pc_alt)→ 快捷手册。
- **流转**:选 TAB → 点玩法卡片「前往」→ 传送确认「是否立即传送?」→ 点「确认」→ 传送到该玩法入口。
- **退出**:esc / 左上返回 → 大世界。

**TAB 切换**(顶部,y~161):目标(769)/ 日常(955)/ 训练(1142)/ 作战(1330)/ 战术(1514)。

## 识别特征(稳定锚点)

- **TAB 文字**:目标 / 日常 / 训练 / 作战 / 战术(顶部,稳定;TAB-XX text area)。
- **QUICK ... HANDBOOK**(左上标题)。
