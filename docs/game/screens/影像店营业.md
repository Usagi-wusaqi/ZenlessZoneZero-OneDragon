# 影像店营业(录像店经营)

录像店「Random Play」日常经营画面(`random_play` app)。**纯经营玩法(无战斗)**:选宣传员 → 上架录像带 → 开始营业。

## 入口 + 状态流转

大世界 → `Transport` → `interact(F)` → (阿甘对话「查看经营状况」) → **影像店营业-经营状况** → 选宣传员 / 上架录像带 → 开始营业 → 返回大世界。

Transport 2 点(`RandomPlayTransportPoint`):
- **POINT_1 录像店-柜台**:传送后 `turn_to_angle(0)` 转正东 + `move_w` 前移 + interact。
- **POINT_2 澄辉坪-录像店营业点** ⭐:传送落点已正对入口,直接 interact(最简,推荐)。

interact 后阿甘 NPC 对话(「嗯呢,嗯呢呢?录像店一切顺利,要查看具体的经营状况吗?」)→ 点「查看经营状况」进经营画面。

流转(`RandomPlayApp` `@operation_node` 链):
传送 → 移动交互 → 等待经营画面加载(昨日账本关闭 / 经营状况 / 查看经营状况) → 识别营业状态(`正在营业`→关闭页面;否则继续) → 点宣传员入口 → 选宣传员 → 识别录像带主题 → 点录像带入口 → 识别推荐上架 → 上架筛选(选主题→上架) → 返回 → 开始营业 → 确认营业 → 营业后确认 → 返回大世界(`BackToNormalWorld`)。
