# 大世界(Overworld)

## 何时出现
登录成功 → 加载画面 → **大世界**。游戏最开放自由的状态,控制角色在 3D 场景中移动、交互 NPC、接任务、进入各种玩法。进游戏后的**主画面**。

screen_info 名为 `大世界-普通`(`-普通` 表示大世界有多个状态变体)。

## 识别特征(稳定锚点)
- **右上角 UI 图标**(模板匹配):快捷手册 @(1531,60) / 功能导览 @(1801,53) / 预备编队 @(1321,51) / 按钮-信息 @(1738,928)。
- **活动入口**(上方横向卡片):零号空洞 / 式舆防卫战。
- **小地图**(右上角,圆形)。
- **底部快捷键**:相机(Tab) / 快捷轮盘 / F1-F4。
- **UID**(右下)。

## 可交互元素
- **左上角菜单按钮**(汉堡「≡」图标,`按钮-菜单` area @96,34-164,108,goto 菜单):打开暂停菜单(底部 仓库/邮件/更多/绳网…),是进入各功能子画面的枢纽。PC 端点击需 `pc_alt=true`(Alt 解锁光标)+ 非零 `press_time`(框架默认 0.1)。
- **活动入口卡片**:零号空洞、式舆防卫战等终局玩法(NEW! 徽章 = 有未查看新内容)。
- **任务导航**:前往任务目标(黄色箭头)。
- **右上角 UI**:快捷手册、功能导览、预备编队。
- **角色自由移动**:键盘/手柄控制角色在 3D 场景中走动。
- **快捷键**:F1-F4(活动快捷进入)、Tab(快捷轮盘)、相机。

## 条件弹窗
- **玩法恢复提示**:上次退出时在某玩法中(如迷失之地/特遣调查),进大世界弹出「当前正在处理:特遣调查是否继续?」+ 取消/确认(_1783247977758)。**模态覆盖**,非独立画面。特遣调查是迷失之地(Lost Void)玩法的一种。
