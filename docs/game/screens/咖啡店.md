# 咖啡店(每日咖啡增益)

每日点咖啡获取战斗增益(体力/经验/掉落)`coffee` app。**核心非战斗(点咖啡)**,但可选挑战副本(含战斗,边界 app)。

## 入口 + 状态流转

大世界 → `Transport` → `interact(F)` → 咖啡店 → 选咖啡 → 点单 → 电量确认 → 选择前往(挑战副本 / 不挑战)→ 返回大世界。

Transport 2 点(`CoffeeTransportPoint`):
- **POINT_1 六分街-咖啡店**:传送后 `turn_to_angle(180)` 转正西 + `move_w` + interact → 咖啡店画面(直接选咖啡)。
- **POINT_2 澄辉坪-汀曼咖啡** ⭐:传送落点正对入口,直接 interact → **对话点单**(汀曼大师对话框选咖啡)。

流转(`CoffeeApp` `@operation_node` 链):
传送 → 等待大世界/咖啡店加载 → 移动交互 → (POINT_2 对话点单:`对话框标题-汀曼大师` → 右侧选项选咖啡 / `对话框-明天再来` 已喝过) → 选择咖啡 → 点单 → (不占用确认 / 点单后跳过)→ 电量确认 → 选择前往(挑战 `CombatSimulation`/`AreaPatrol`/`ExpertChallenge` / 不挑战-对话框确认)→ 返回大世界。
